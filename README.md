# Псковские угодья

Одностраничный сайт охотничьего хозяйства. Вёрстка по макету: тёмно-зелёный + кремовый, serif-заголовки.

## Запуск

```bash
python3 -m http.server 8080
# http://localhost:8080
```

## Замена телефона

Найти и заменить `+7XXXXXXXXXX` и `+7 (XXX) XXX-XX-XX` в `index.html`.

## Свои фото

1. Положите файлы в `images/`
2. Замените URL Unsplash в `index.html` и фон hero в `css/styles.css` (`.hero__bg`)
3. Референс макета: `images/mockup-reference.png`

## Соцсети

В подвале замените ссылки `https://t.me/` и `https://wa.me/` на реальные.

## Деплой

GitHub Pages, Netlify Drop или любой статический хостинг с HTTPS.
